# 📊 Data Professional Survey Breakdown

## 📌 Overview
Analysis and visualization of a real-world survey conducted among data professionals
worldwide, exploring salaries, job roles, favorite tools, and work-life satisfaction levels.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX

## 📷 Dashboard Preview
![Dashboard](screenshots/dashboard_preview.png)

## 📁 Files
- `Data_Professional_Survey.pbix` — Main Power BI file
- `survey_data.xlsx` — Raw survey data

## 📊 Key Insights
- **Python** is the most popular programming language among data professionals
- **Data Scientists** earn the highest average salary compared to other data roles
- Majority of respondents rated **breaking into data** as neither easy nor difficult
- Average **work/life balance** satisfaction score was around **5.74 / 10**
- Average **salary satisfaction** score was around **4.27 / 10**
- **United States** had the highest number of survey respondents
- Most data professionals transitioned into the field from a **different industry**

## 📂 Dataset
- Total Survey Respondents: **630**
- Data collected from real data professionals across the globe
